## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Requirements
- node > 10
- Have a relational database running (preferably postgres)

## Configuration
Copy `ormconfig.example.js` to `ormconfig.js` and modify it with your local db settings.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
